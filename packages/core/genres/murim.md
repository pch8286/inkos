## 장르 금기

- 무공, 경지, 내공의 차이가 장면에서 체감되지 않는 것
- 말로만 강호를 말하고 실제 강호의 질서와 체면이 없는 것
- 복수와 은원, 문파와 사제 관계가 사건을 밀지 않는 것
- 기연이 너무 자주 나오거나, 아무 대가 없이 주어지는 것
- 초식 설명만 길고 손맛, 호흡, 거리감이 없는 것

## 서사 규칙

- 강함은 숫자보다 기세, 호흡, 시선, 손끝의 압박으로 먼저 드러낸다
- 경지는 단순한 강약이 아니라 선택과 태도의 변화로도 표현한다
- 문파, 강호, 사제 관계, 은원은 매 장면의 동력으로 유지한다
- 대결은 승패뿐 아니라 상대의 체면, 정보, 빚, 인맥을 건드려야 한다

## 문체 규칙

- 문장은 단단하고 직선적으로 가져가되, 결정적 순간에는 호흡을 늦춘다
- 과장된 수사보다 검, 손, 발, 피로, 뼈, 숨 같은 물성을 우선한다
- 무공 이름은 멋보다 기능이 먼저 와야 한다
