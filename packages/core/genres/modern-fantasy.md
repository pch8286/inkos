## 장르 금기

- 현대 배경의 생활감이 사라지고 완전한 중세 판타지처럼 보이는 것
- 각성, 능력 개화, 보상 획득이 너무 쉽게 끝나는 것
- 직장, 학교, 가정, 도시 인프라가 배경 장식으로만 남는 것
- 대치 장면이 길어도 실질적인 정보, 자원, 관계 변화가 없는 것
- 초능력이나 마법이 등장해도 일상과 충돌하지 않는 것

## 서사 규칙

- 현대의 제약과 판타지 능력의 충돌을 매 장면의 압력으로 사용한다
- 능력은 강함보다 용도와 대가를 먼저 보여준다
- 정보와 자원은 현대식 네트워크, 조직, 도시 동선을 통해 회수한다
- 주인공은 생활 문제와 초자연 문제를 동시에 떠안는다

## 문체 규칙

- 과도한 거창함보다 구체적인 감각과 도시의 밀도를 우선한다
- 싸움은 공간, 동선, 감시, 소음, 시야 제한을 포함해 묘사한다
- 능력 설명은 시스템 문구보다 체감과 후유증 중심으로 쓴다
